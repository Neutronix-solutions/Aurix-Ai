### 1. ✅ Authentication Flow
- Register → Login → Logout

- Session persistence

### 2. ✅ Chatbot Feature
- Chat UI visibility (NOT hidden behind navbar)
- Send/receive messages
- Empty message handling
- Long message scrolling behavior
- Network failure handling

---

### 3. ✅ Currency System
- Change currency in settings
- Verify update reflects in:
  - Home page
  - Markets page
  - Gold page
- Test multiple currency switches rapidly
- Validate data consistency

---

### 4. ✅ Gold Feature (Disabled Flow)
- Attempt to purchase gold
- Verify message appears:
  "هذه الميزة غير متاحة حالياً..."
- Ensure:
  - No API call is triggered
  - No crash occurs
  - UI remains stable

---

### 5. ✅ Navigation & Routing
- All screens accessible
- No broken routes
- Back navigation works correctly
- Protected routes blocked after logout

---

### 6. ✅ UI/UX Testing (Mobile Focus)
- Test on different screen sizes
- Check:
  - Overlapping elements
  - Hidden buttons
  - Scroll issues
- Keyboard does not block inputs
- Bottom navbar does not hide content

---

### 7. ✅ API & Data Handling
- Simulate:
  - Slow network
  - API failure (500 / timeout)
- Ensure:
  - Proper error messages
  - No crashes
  - Loading states exist

### 8. ✅ Performance Testing
- App launch time
- Screen load time
- Chat responsiveness
- Avoid unnecessary re-renders

---

### 9. ✅ Security Checks

### 10. ✅ Build & Deployment Readiness
- No runtime errors

- Clean production build

  - Android build (APK/AAB)
